# stock-console

控制台实时显示股票信息

为什么要做控制台程序而不是只做web？

1. web应用开发耗时太长
2. 控制台程序对快捷键很友好
3. 小巧灵活，自由度高
4. 计算成本低，可移植性强

---

## 当前版本（v0.1.0）

* &#x2705; 实现控制台实时显示股票基础信息

```shell
# 每500毫秒更新一次价格信息
java -jar stock-console.jar 500 QQQ SPY
```
