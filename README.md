# stock-console

控制台实时显示股票信息

为什么要做控制台程序而不是只做web？

1. web应用开发耗时太长
2. 控制台程序对快捷键很友好
3. 小巧灵活，自由度高
4. 计算成本低，可移植性强

---

## 当前版本（v1.0.0）

JDK版本：17+

* &#x2705; 库版本升级
* &#x2705; 更新控制台指令模块
* &#x2705; 代码重构

```shell
# 默认每500毫秒更新一次价格信息，-h -p 指定网络代理
java -jar .\stock-console.jar -s QQQ,SPY,AAPL,000001.SS -h 127.0.0.1 -p 10809
```
